这是一个 slidev addon 项目。我们已经设计了若干 addon。接下来的需要，是在已经实现的 slidev-addon-card 基础上，再实现一个 barrel 的 layout,可参照 slidev-addon-quantide-layout 下的 cards.vue.

我们要实现的 addon 是 addons/slidev-addon-quantide-layouts, barrel.vue 是其中的一个 layout.

与 cards 主要不同在于，barrel 将提供令人惊艳的3D 视觉效果。

## 1. 输入

用户的输入是：

```md
---
layout: barrel
gendre: polygon
diameter: 600
margin: 200
width: 300
clicks: 5
background: https://picsum.photos/1920/1080?nature
---

## Barrel Card 1

- icon: fa-rocket
- orientation: portrait
- content: Three.js barrel mode card
- width: 300px

## Barrel Card 2

- icon: fa-star
- orientation: portrait
- content: Three.js barrel mode card
- width: 300px

## Barrel Card 3

- icon: fa-heart
- orientation: portrait
- content: Three.js barrel mode card
- width: 300px
```


gendre 表明 barrel 的形状，可以是多面体(polygon)，也可以是圆柱体(circle)
diameter 是 barrel 的直径，单位是像素。
margin 是卡片与卡片间的间距，中间透明填充。单位是像素
width 是卡片的宽度，单位是像素。卡片的高度是 5/3倍
clicks 是点击次数，默认是 5 次。
background 是背景图片，可以是图片地址，也可以是颜色。

## 2. 卡片的生成

1. 安排一个cards-container 容器。在 mount 时，解析本 layout 中的 slot文本内容，生成若干个 Card。Card 负责样式和渲染。如何解析 slot 文本，以了解它定义了多少个 Card 呢？这需要参考 slidev-addon-card/components/card.vue的代码。我们**必须**保持一致性的格式定义和解析。

cards-container 应该对用户不可见，但要能截图（html2canvas或者 html2img）
2. 将$fontmatter.width 应用于生成的 Card. Card 的样式设置为 portrait

3. 对n 个卡片进行截图（可调用 html2canvas 或者 html2img，请择优选择），保存为一个数组。记截图的宽为 img_width，高为 img_height
4. 为了调试，将截图下载到本地。

## 3. polygon风格

1. 生成一个边长为 img_width 的正n 边形
2. 将第2步（卡片的生成）中，每个 Card 的截图贴到正 n 边形的每一边上
3. 旋转这些截图，使得它们与正 n 边形相垂直。现在，正 n 边形在 xy 平面上，而每个卡片都沿 z轴指向观察者
4. 沿 x轴旋转正 n 边形及连接的截图90度。
5. 将生成的正 n 面体缩放，使得直径为 diameter 的圆柱形刚好成为它的外接圆。

## 4. circle 风格

1. 以 n (卡片数) * （img_width + margin）为周长，生成一个透明圆柱体。
2. 将第2步（卡片的生成）中，得到的每个Card 的贴图贴到圆柱体上。每个贴图中间，插入一个与img_height 等高，宽度为 margin 的透明区域
3. 将贴图后的对象，缩放为直径为 diameter 大小的圆柱体。
