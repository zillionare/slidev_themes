# Slidev Addon Rehearsal

一个专为 Slidev 设计的彩排录制与时长预估插件，帮助您精确控制幻灯片的自动播放节奏。

## 功能特性

- **实战录制**：在演示模式下自动记录每一页、每一个 Click 的停留时间。
- **AI 时长预估**：基于演讲者备注（Notes）中的台词字数和标点权重，一键生成预估播放时长。
- **智能权重计算**：
    - 普通汉字/字符：权重 1
    - 逗号 (，/ ,)：权重 2
    - 句号 (。/ .)：权重 4
- **内存同步**：编辑后的时间可即时同步到内存 Frontmatter，预览立即生效。
- **快速保存**：双击 Slide 标题即可将该页的 `autoPlay` 和 `audioStart` 同步回 slide。
- **后退检测**：支持在彩排过程中后退，自动重置当前页记录。

## 安装使用

1. 在您的 Slidev 项目中安装此 Addon。
2. 在主幻灯片的 `frontmatter` 中启用：

```yaml
addons:
  - slidev-addon-rehearsal
```

3. 进入 **Presenter Mode (演讲者模式)**，点击左下角的计时器图标或使用快捷键 `Alt + R` 打开面板。

## 配置项

您可以在全局 `frontmatter` 中配置以下参数：

| 参数              | 说明                          | 默认值 |
| ----------------- | ----------------------------- | ------ |
| `wordsPerMinute`  | 预估时长时的语速（字/分钟）   | `250`  |
| `rehearsalBuffer` | 每个 Click 后的缓冲时间（秒） | `0.5`  |

## 演讲者备注规范 (用于 AI 预估)

在您的 Slide 底部使用 `<!-- -->` 编写备注，并使用 `[click]` 标签分隔不同步骤的台词：

```markdown
---
layout: default
---

# 欢迎来到我的演讲

<!--
这是 Click 0 (首页进入) 的台词。
[click]
这是第一个点击后的台词。
[click]
这是第二个点击后的台词。
-->
```

## 简单示例

假设您的备注内容为：
`这是雪中的光谷中心，宛如流浪地球中的行星发动机。[click] 匡醍量化，就诞生在这里`

点击 **“生成预估时间 (AI)”** 后，系统将自动计算：
- 台词 0 (22字 + 1逗号 + 1句号)：`(28 / 250) * 60 + 0.5 = 7.22s`
- 台词 1 (10字 + 1逗号)：`(12 / 250) * 60 + 0.5 = 3.38s`

生成结果如下并可直接保存或复制：
```yaml
audioStart: 0
autoPlay: [7.22, 3.38]
```

## 操作指南

- **双击标题**：将该页配置保存到内存。
- **保存全部**：将所有页面的彩排数据同步到内存，预览立即生效。
- **手动微调**：点击面板上的时间数组或 `audioStart` 数值可直接编辑，失焦后自动同步。
- **Alt + R**：快速切换彩排面板显示/隐藏。
