## [1.0.0] - 2024-12-19

### Added
- Initial release of slidev-addon-card
- Card component with image, title, and content support
- Multiple orientation options (landscape/portrait)
- Various frame styles (rectangle, circle, ellipse, diamond, gem)
- Customizable styling options:
  - Background and border colors
  - Title and text colors
  - Border radius
  - Drop shadow toggle
- Responsive design with mobile adaptation
- Hover effects and smooth animations
- Comprehensive documentation and examples
- TypeScript support with prop validation

### Features
- **Image Frames**: Support for 5 different frame styles including unique gem shape
- **Flexible Layout**: Landscape and portrait orientations with automatic mobile adaptation
- **Custom Styling**: Full control over colors, borders, and visual effects
- **Accessibility**: Proper alt text support and semantic HTML structure
- **Performance**: Optimized CSS with hardware acceleration for smooth animations

### Documentation
- Complete README with usage examples
- Interactive example slides demonstrating all features
- Property reference table
- Styling tips and best practices
